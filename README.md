Project Overview 
Built an Online Furniture Mart is a multi-user e-commerce web application developed using ASP.NET Web Forms. 
The application allows users to browse furniture products by category, add items to cart, place orders, and complete payments.
The system includes separate Admin and User modules and follows a 2-layer architecture for structured development.
The payment functionality is handled via a separate WCF service to simulate a real world service-oriented architecture.
Technologies Used:Front End-ASP.NET Web Forms,HTML,CSS  Backend-C#,ADO.NET Database-SQL Server
Server Controls (GridView, DataList, DropDownList, TextBox, Button)
Service Integration-WCF (Windows Communication Foundation) for Payment Module
User Module Features
   User Registration with Validations
   User Login Authentication
   Browse Products Category-wise
   Products displayed using DataList Control
   Add to Cart functionality
   Place and Confirm Orders
   Payment Integration using WCF Service
Admin Module Features
   Admin Registration and Login
   Add and Edit Categories and Products
   Product Management using GridView
   Used CommandField and Template Field for Edit/Delete operations
   View Registered Users
   Block / Unblock Users
   View User Feedback
   Reply to User Feedback
Key Functionalities
   Multi-user authentication system
   Form validations
   CRUD operations (Create, Read, Update, Delete)
   Session management
   Category-based product filtering
   Cart management
   Order confirmation workflow
The UI design of this project is based on a pre-built template source.
The template was customized and integrated into the application and the whole backend was implemented by me.
